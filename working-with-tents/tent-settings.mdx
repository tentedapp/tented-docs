## Tent Settings Overview

Tent settings allow you to customize the behavior, appearance, and functionality of individual tents. These settings are specific to each tent and don't affect your global workspace preferences.

## General Settings

### Basic Information

**Tent Name:**
- Display name for your tent
- Used in dashboard and navigation
- Can be changed at any time
- Helps with organization and identification

**Description:**
- Brief description of tent purpose
- Used for internal reference
- Helps team members understand tent goals
- Visible in tent details and reports

**Status Management:**
- **Draft**: Private, not published
- **Published**: Live and publicly accessible
- **Archived**: Retired from active use
- **Unpublished**: Was published, now private

### Visibility Settings

**Access Control:**
- **Public**: Accessible to anyone with the URL
- **Private**: Requires authentication to view
- **Password Protected**: Requires password to access
- **Team Only**: Visible only to team members

**Sharing Options:**
- Enable/disable social sharing
- Control preview link sharing
- Manage embed permissions
- Set expiration dates for access
